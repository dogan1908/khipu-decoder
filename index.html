About this tool

This tool brings together two distinct bodies of evidence about the pre-Columbian Andean world: the corpus of khipus — knotted cord records used by the Inca and earlier Andean civilizations — and the lexicon of Quechua, the language family spoken across the Andes from before the Inca expansion to the present day. The 629 khipus included here are drawn from the Open Khipu Repository, the public dataset assembled by the Khipu Database Project at Harvard University.

The decoder operates on a phonosemantic gematria hypothesis: each digit 0–9 is mapped to a class of Quechua consonants according to articulatory features (nasals, uvulars, liquids, laterals, labials, dentals, sibilants, plosives, aspirates, ejectives). A multi-digit cord value such as 75 is read as a sequence of consonant-vowel syllables (P+a · M+a → puma, "mountain lion") and matched against a curated database of Quechua words categorized into 15 semantic fields (water, plant, animal, body, human, spirit, tool, place, time, number, action, quality, food, weather, social).

Features

  • Khipu browser — 629 khipus searchable by ID, provenance, museum, OKR number, Quechua word or numeric value. Search results flag khipus containing matching values with ⚡N counters.
  • Per-cord gematria decoding — each cord value is broken into digits, mapped to consonant classes, expanded into all syllable combinations, and matched against the word database with category badges.
  • Word browser by category — 557 Quechua words filtered by combinable category and free-text search. Each word lists every khipu containing its gematria value as clickable chips.
  • Khipu-aware mindmap — radial SVG visualization. When a khipu is selected, the mindmap shows only the words whose values occur in that khipu, with cluster sizes scaled by occurrence frequency.
  • Three gematria systems — Quechua phonosemantic (Balban 2026, UR281-verified), Katapayadi (Sanskrit), and a fully editable custom system.
  • Structural analysis — automatic detection of repeated marker values, Σ-checksum candidates, and group boundaries within each khipu.
  • Self-contained, offline-capable — single HTML file, no build step, no framework, no backend. Runs entirely in the browser.

Quechua gematria system

The phonosemantic mapping (Balban 2026): each digit corresponds to a class of Quechua consonants grouped by place and manner of articulation.

0N · ÑNasal → emptiness, negation
1K · Q · YUvular / velar / semivowel
2R · HLiquid + breath
3L · LlLateral → flow
4WLabial semivowel → wind
5MLabial nasal → mother / earth
6T · Ch · SDental / palatal / sibilant
7PLabial plosive
8Kh · Qh · Ph · Th · ChhAspirates
9K' · Q' · P' · T' · Ch'Ejectives → compressed flow

Worked example: cord value 75 → digits [7, 5] → consonants [P, M] → syllables {pa-ma, pu-ma, pi-mi, …} → match: puma ("mountain lion") in 50 khipus across the corpus.

Data sources

Khipu corpus. The 629 khipus and their 40,970 cord values are derived from the Open Khipu Repository (Harvard Khipu Database Project), a longstanding effort by Gary Urton, Carrie Brezine and collaborators to digitize and standardize records from museum collections worldwide. Cord values are extracted from the public SQLite distribution.

Quechua wordlist. 557 words compiled from standard Cuzco-Collao Quechua reference dictionaries; gematria values computed using the phonosemantic system above. Categorization assigned by keyword heuristics over English glosses, then manually refined.
